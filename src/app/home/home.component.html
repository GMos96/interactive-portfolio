AngularJS to Angular
I lead a team of engineers to design a framework/process around moving from a AngularJS SPA to a hybrid SPA using Angular.
What Steps Did I Take?

Documented and designed our migration strategy. I decided on a multi-step strangler pattern where new development would be done in Angular, and AngularJS would remain our shell and handle routing, authentication, and other high-level functionality, like navigation, system preference retrieval, etc.

Feature work in the Angular project is done in libraries, with multiple "shells." We have a shell for our new Angular application (to be completed when migration is complete) and a web component shell with needed provider overrides. This web component shell uses Angular Elements to create a router component that handles retrieving route information from AngularJS and relaying it to Angular.

Implemented our AngularJSResourceService, which uses the existing AngularJS injector to reuse existing AngularJS services for core functionality that must be shared. This is critical for authentication, routing, as well as utilizing massive features that aren't planned to be moved until the end of the project.
